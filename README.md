1.Сгенерировать один пароль
    -	Настройка длины (8-50 символов)
    -	Выбор использования цифр и специальных символов
    -	Автоматическая оценка сложности
    -	Возможность копирования в буфер обмена
2.Сгенерировать несколько паролей
    -Генерация от 1 до 20 паролей
    -Настройка параметров для всех паролей
    -Возможность сохранения в файл
3.Пароли по уровню сложности
    -Слабый: 8 символов, только буквы
    -Средний: 12 символов, буквы + цифры
    -Сильный: 16 символов, все символы
    -Очень сильный: 20 символов, все символы
4.Сохранение в файл
    -Сохранение с нумерацией
    -Указание длины и сложности каждого пароля
    -Поддержка пользовательского имени файла
